### 5. 초기 설정 (`application.properties`, `SecurityConfig`, `HealthController`)

**application.properties 설정:**
```properties
spring.application.name=pnn-backend
server.port=8080
spring.datasource.url=jdbc:postgresql://localhost:5432/pnn-db
spring.datasource.username=postgres
spring.datasource.password=1234  ← 추후 .env 분리 예정
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
```

**SecurityConfig:** CSRF 비활성화, 전체 permitAll (개발 초기), HTTP Basic 활성화
→ JWT 구현 시 `.anyRequest().authenticated()`로 변경 예정

**HealthController:** `GET /api/health` → `{ "status": "ok" }` 브라우저 확인 완료

**패키지 구조:**
```
com.pnn.backend
  ├─ config/       (SecurityConfig)
  ├─ controller/   (HealthController)
  ├─ service/
  ├─ domain/
  ├─ repository/
  └─ client/       (Python AI 호출용, 향후)
```
